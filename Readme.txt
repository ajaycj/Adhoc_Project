Energy Efficient Communication of Wireless Ad- Hoc Networks
--------------------------------------------------------------

Contents:
1. Java Files : sink_node1.java for sink node
	        node1.java for other nodes
2. Topology Files
3. Other files used for documentation and logs.
4. Batch Commands.

Running the Project:
1. Extract the contents of the rar file as it is.
2. Whenever ready, run the project1.bat file.
3. This will open 20 command windows wherein the sink node and other nodes will get initialized and connected .
4. Find the sink node command window and input the query (min | max | avg).
5. The result as well as each activity will be recorded inside log files given in log folder.

OS : Only Windows OS (because of the batch files).
